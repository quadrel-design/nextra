# Пользовательские идентификаторы заголовков

Вы можете указать пользовательские идентификаторы заголовков, используя формат `## Header [#custom-id]`. Например:

```markdown
## Foo [#bar]
```

## У этого есть пользовательский идентификатор [#пример-пользовательского-идентификатора]

Вы можете увидеть пользовательский идентификатор заголовка в заголовке выше. Обратите внимание, как при наведении указателя мыши на заголовок, так и в оглавлении используется ссылка `пример-пользовательского-идентификатора`.

## У этого нет пользовательского идентификатора

Этот заголовок не имеет пользовательского идентификатора. Идентификатор генерируется из текста заголовка.
