# `<Search>` Component

|               |                               |                                                                           |
| ------------- | ----------------------------- | ------------------------------------------------------------------------- |
| emptyResult   | `string \| ReactElement{:ts}` | Not found text. Default: `"No results found."{:json}`                     |
| loading       | `string \| ReactElement{:ts}` | Loading text. Default: `"Loading…"{:json}`                                |
| errorText     | `string \| ReactElement{:ts}` | Error text. Default: `"Failed to load search index."{:json}`              |
| placeholder   | `string{:ts}`                 | Placeholder text. Default: `"Search documentation…"{:json}`               |
| className     | `string{:ts}`                 | CSS class name                                                            |
| searchOptions | `PagefindSearchOptions{:ts}`  | `{preload?: boolean; verbose?: boolean; filters?: object; sort?: object}` |
