# `<Search>` Component

|               |                               |                                                                                 |
| ------------- | ----------------------------- | ------------------------------------------------------------------------------- |
| emptyResult   | `string \| ReactElement{:ts}` | Not found text. Default: `'No results found.'{:ts}`.                            |
| loading       | `string \| ReactElement{:ts}` | Loading text. Default: `'Loading…'{:ts}`.                                       |
| errorText     | `string \| ReactElement{:ts}` | Error text. Default: `'Failed to load search index.'{:ts}`.                     |
| placeholder   | `string{:ts}`                 | Placeholder text. Default: `'Search documentation…'{:ts}`.                      |
| className     | `string{:ts}`                 | CSS class name.                                                                 |
| searchOptions | `PagefindSearchOptions{:ts}`  | `{preload?: boolean; verbose?: boolean; filters?: object; sort?: object}{:ts}`. |
